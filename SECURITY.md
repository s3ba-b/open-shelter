# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ShelterStack, please report it
privately. **Do not open a public issue.**

- Email: **seb.bobrowski@proton.me**
- Alternatively, use GitHub's
  [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
  if enabled on this repository.

Please include:

- A description of the vulnerability and its impact
- Steps to reproduce
- Any suggested remediation, if known

You can expect an initial response within a reasonable timeframe. Once the issue
is confirmed and fixed, we will coordinate disclosure.

## Supported Versions

This project is in active development. Security fixes are applied to the latest
`main` and the most recent release.
